🚗 Global EV Sales Analysis (Power BI Project)

📊 Project Overview

This project analyzes global Electric Vehicle (EV) data from 2010 to 2024 using Power BI. It provides insights into EV growth trends, regional adoption, powertrain distribution, and energy usage.

🔧 Tools Used

* Power BI
* Data Analysis
* Data Visualization

📁 Project Files

* 📄 Full Dashboard (PDF)
* 📊 Power BI File (.pbix)
* 📂 Dataset (.csv)
  
📌 Key Insights

* China leads global EV adoption
* BEVs dominate the market (~80%)
* EV growth accelerated significantly after 2020
* Charging infrastructure is rapidly expanding

🚀 Features

* Multi-page interactive dashboard
* KPI cards and filters (Year, Region, Powertrain)
* Trend and comparative analysis
